Download

Download sweetblue version 0.1.4 in .zip format.

Installation

Unzip and put the extracted sweetblue folder into the libraries folder of your processing sketches. Reference and examples are included in the sweetblue folder.

Keywords processing, android, bluetooth

Reference. Have a look at the javadoc reference here. a copy of the reference is included in the .zip as well.
